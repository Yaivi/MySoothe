# MySoothe (Basic Layouts in Compose):
Lo veo como un avance muy útil con respecto a la introducción a elementos Composable, que explica más en profundidad diferentes formas de hacer layouts con Columns, Row, etc, a través de Modifier. El como alinear, organizar y juntar diferentes elementos a través componentes Material, como Scaffold, y sobre todo el adaptar todos los elementos a diferentes dimensiones de pantallas. Cada una de estos elementos que enseña a crear y modificar serán muy importantes a la hora de desarrollar la aplicación para que sea lo más profesional posible.
